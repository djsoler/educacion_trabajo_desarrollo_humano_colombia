como abro conda y hago Crear y activar el ambiente virtual con Conda

Para abrir Conda y crear y activar un ambiente virtual, sigue estos pasos:

Abrir Conda

  1. En Windows:

    • Abre la aplicación "Anaconda Prompt" desde el menú de inicio. Esta es una línea de comandos que viene con Anaconda.

  2. En macOS o Linux:

    • Abre la aplicación "Terminal".

Crear y Activar el Ambiente Virtual

Una vez que estés en la línea de comandos (Anaconda Prompt o Terminal), sigue estos pasos:

  1. Crear un nuevo ambiente virtual:

    Ejecuta el siguiente comando para crear un ambiente virtual llamado myenv con Python 3.10:

    bash
    conda create --name myenv python=3.10
    • myenv es el nombre del ambiente. Puedes cambiarlo por cualquier otro nombre que prefieras.
    • python=3.10 especifica la versión de Python que deseas usar.

    Durante la creación del ambiente, Conda te pedirá confirmación para instalar las dependencias necesarias. Escribe y y presiona Enter para continuar.

  2. Activar el ambiente virtual:

    Después de crear el ambiente, actívalo con el siguiente comando:

    bash
    conda activate myenv
    • Una vez activado, deberías ver el nombre del ambiente (myenv) al inicio de la línea de comandos, lo que indica que estás trabajando dentro de ese ambiente.

Notas adicionales

  • Si deseas ver una lista de todos los ambientes que has creado, puedes usar el comando:

    bash
    conda env list

  • Para desactivar un ambiente y volver al ambiente base, usa el comando:

    bash
    conda deactivate
